Dev Tools (Google Chrome)

  1. Способы открытия dev tools
    • Горячие клавиши: Ctrl+Shift+I (Windows) или Cmd+Opt+I (Mac Os)
    • Правой кнопкой мыши на любой элемент веб-страницы: проинспектировать элемент, либо исследовать элемент.
    • В меню вид, либо где-то в настройках поискать поле для разработчиков (Developer) или разработка. Там выбрать Developer Tools.
